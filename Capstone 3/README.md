# Metropolitan Business Feasibility Prediction System

A data pipeline, machine-learning suite, and interactive BI dashboard for ranking
and predicting high-opportunity U.S. metropolitan areas across **9 broad business categories**.

---

## Quick start

```bash
# Install dependencies
pip install -r requirements.txt

# Regenerate all outputs, CSVs, charts, and reports
python3 main.py

# Launch the interactive BI dashboard
streamlit run app.py
```

Optional: limit to top N metros by population (default = all 386)
```bash
METRO_SAMPLE_LIMIT=200 python3 main.py
```

## Data sources

| Dataset | Year | Provider | Coverage |
|---------|------|----------|----------|
| County Business Patterns (CBP) | **2023** | U.S. Census Bureau | 369,496 rows, 925 metro/micro areas |
| American Community Survey (ACS) | 2024 | U.S. Census Bureau | 393 metro areas, income + population |
| BLS Metro Unemployment | 2020–2024 | Bureau of Labor Statistics | Annual unemployment, 5-year history |
| **BEA Regional Price Parities (RPP)** | **2024** | Bureau of Economic Analysis | All-items + Housing index per MSA |

**Final analysis sample:** 386 overlapping metropolitan areas across 51 state/district labels.

## Feature engineering — 34-feature MODEL_FEATURES set

| Feature group | Features |
|---|---|
| Structural CBP (7) | total emp, est, small/mid/large est share, avg emp per est |
| Payroll & labour (3) | annual payroll per employee, payroll per capita, employment per 100 residents |
| Market context (4) | unemployment trend slope (2020-2024), state business climate score, **cost-of-living index, housing-cost index** |
| Sector employment shares (20) | 2-digit NAICS emp share codes 11 through 99 |

Additional derived columns used in scoring (not model features):
`real_median_income` (= median_income × 100 / cost_of_living_index),
`sector_est_per_10k_{code}`, `saturation_{code}`, `unemployment_2020-2023`.

## Opportunity score (transparent weighted index)

| Component | Weight |
|---|---|
| **Real median income** (PPP-adjusted via BEA RPP) | 25% |
| Inverse unemployment rate | 20% |
| Annual payroll per employee | 15% |
| Establishments per 10,000 residents | 15% |
| Industry entropy (business diversity) | 15% |
| **Affordability** (inverse cost-of-living index) | 10% |

Top 30% by opportunity score = **high-opportunity** (classification target).

The cost-of-living adjustment ensures that a high-income metro with $4,200/month rent
doesn't outrank a moderate-income metro with $1,200/month rent for cost-sensitive
small businesses. After this correction:
- San Francisco dropped from #2 → #4
- Boston dropped from top-5 → #13
- Midland TX, Sioux Falls, Bismarck moved into the top 6

## Models

### Classification — predicting high-opportunity metros
- **Candidates:** LogisticRegression, RandomForestClassifier, GradientBoostingClassifier (tuned), VotingClassifier (soft, all 3)
- **Selection:** best validation F1
- **Tuning:** GridSearchCV on the winning model (5-fold CV, F1 scoring)
- **Calibration:** CalibratedClassifierCV (isotonic) for well-calibrated probabilities

### Regression — predicting unemployment rate
- **Candidates:** LinearRegression, RandomForestRegressor, GradientBoostingRegressor
- **Target:** 2024 annual metro unemployment rate

## Business feasibility scoring

```
Business Feasibility Score = 70% × Market Strength + 30% × Category Fit

Market Strength = calibrated classifier probability
Category Fit    = 60% × sector employment share + 40% × sector establishment density
```

## Analytics extensions

### K-Means metro clustering (k=5)
Metros grouped by economic fingerprint, visualised in PCA 2-D space.
Cluster archetypes: Elite Opportunity Market, Affluent Low-Risk Hub, Major Metro Engine,
Business-Dense Market, High-Unemployment / Value Market.

### Sensitivity analysis (5 weight scenarios)
Ranks metros under: Income-Heavy, Employment-Heavy, Balanced (default), Density-Heavy, Diversity-Heavy.
Metros labelled **Stable** (rank_std ≤ 15), **Moderate**, or **Volatile** (rank_std > 35).

## Limitations

- CBP is 2023; ACS, BLS, and BEA RPP are 2024 (intentional trade-off for data availability).
- Metro-level feasibility — not a business-level causal model.
- Opportunity score is a decision-support tool, not an observed outcome.
- ✅ ~~Housing cost / rent data not yet included~~ — added in Phase 5 via BEA RPP.
