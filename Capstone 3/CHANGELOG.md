## Phase 5 — Cost-of-Living, Data Refresh, Submission Polish (2026-05-03)

### Submission deliverables
- **Executive PowerPoint deck** — `outputs/executive_summary.pptx` (8 widescreen slides:
  title, data stack, KPIs, top-10 metros, business categories, clusters, models,
  feature importance). Auto-regenerated by `python3 main.py`. New module `exec_report.py`.
- **Executive PDF summary** — `outputs/executive_summary.pdf` (one-page text summary,
  pure-stdlib generator, no extra dependency).
- **Per-metro dossier export** — Market Explorer button generates a metro-specific
  PowerPoint dossier (title slide, KPI strip, all 9 business scores).
- **Sidebar download buttons** for both executive deliverables.

### Geographic features
- **Census CBSA gazetteer** — lat/lon centroids for all 935 metros, fetched from
  `www2.census.gov/geo/docs/maps-data/data/gazetteer/2024_Gazetteer/`.
- **Bubble map** — new `px.scatter_geo` visualisation in Executive Overview tab,
  color = opportunity score, size = population.
- `lat` and `lon` exported in `full_metro_ranking.csv`.

### Watchlist / Session bookmarks
- New `st.session_state.watchlist` — save/un-save metros from Market Explorer.
- Sidebar shows the saved list with one-click remove + "Clear all".
- Survives across tabs within a session.



### New data source
- **BEA Regional Price Parities (RPP) 2024** — official cost-of-living and housing-cost
  indices for every U.S. MSA, fetched from `apps.bea.gov`. 100 = US national average.
- File: `metro_data/bea_rpp_msa_2008_2024.csv` (full 2008–2024 history embedded).

### CBP refresh: 2022 → 2023
- Pipeline now auto-selects `cbp23msa.zip` if present, falls back to `cbp22msa.zip`.
- `load_cbp_raw()` now reads any `*.txt` member inside the zip (not hardcoded to a year).
- Total raw rows: 369,496 (vs. 370,118 from 2022).

### New model features (34 total, up from 32)
- `cost_of_living_index` — BEA RPP all-items, 100 = US average
- `housing_cost_index`   — BEA RPP services-housing component
- (derived) `real_median_income` = `median_income × 100 / cost_of_living_index`

### Updated opportunity-score formula
```
0.25 × real_median_income_norm     ← was 25% gross income
0.20 × inverse_unemployment_norm    (was 25%)
0.15 × annual_payroll_per_employee_norm (was 20%)
0.15 × establishments_per_10k_norm
0.15 × industry_entropy_norm
0.10 × affordability_norm           ← NEW (cost-of-living penalty)
```

## Phase 4 — Full Analytics & BI Upgrade (2026-04-28)

### Model & Analytics
- **GridSearchCV tuning** — best classifier is now hyperparameter-tuned after validation
  selection (5-fold CV, F1 scoring); supports GBC, LR, and RF parameter grids.
- **VotingClassifier** — soft-voting ensemble of LR + RF + GBC added as a 4th candidate.
- **CalibratedClassifierCV** — market-strength probabilities are now statistically
  calibrated (isotonic regression, 5-fold).
- **GradientBoostingClassifier** — improved default params:
  n_estimators=300, max_depth=3, learning_rate=0.05, subsample=0.8, min_samples_leaf=5.
- **K-Means metro clustering (k=5)** — metros grouped by economic fingerprint;
  archetypes named algorithmically; PCA 2-D projection saved.
- **Sensitivity analysis** — 5 alternative weight scenarios tested; metros labelled
  Stable / Moderate / Volatile based on rank standard deviation.
- **Sector saturation index** — `saturation_{code}` for each of the 9 business
  category sectors (ratio vs national median establishment density).

### New features added to MODEL_FEATURES (32 total, up from 30)
- `unemployment_trend_slope` — linear slope of unemployment 2020-2024 (negative = improving).
- `state_business_climate_norm` — Tax Foundation 2024 state climate rank, inverted and normalised.

### New data
- BLS unemployment history 2020-2023 now passed through the merge pipeline (previously
  only 2024 and the 2024-2023 delta were retained).

### New business categories (9 total, up from 6)
- `education` — Education / Childcare Services (NAICS 61)
- `construction` — Construction / Trades (NAICS 23)
- `finance` — Finance / Insurance Services (NAICS 52)

### New output files
- `cluster_analysis.csv` — cluster labels, PCA coords, key metrics per metro
- `sensitivity_analysis.csv` — per-metro rank under 5 weight scenarios + stability label
- `cluster_pca_scatter.png` — PCA 2-D cluster scatter
- `unemployment_trend_top10.png` — unemployment trend 2020-2024 for top 10 metros
- `sensitivity_analysis_heatmap.png` — rank heatmap across weight scenarios
- `business_category_correlation.png` — Pearson correlation matrix across 9 business types
- `full_metro_ranking.csv` now includes `cluster_name`, `unemployment_trend_slope`,
  `state_business_climate_norm`.

### Dashboard (app.py)
- **8 tabs** (up from 6): added Cluster Analysis (tab 6) and Sensitivity Analysis (tab 7).
- **Market Explorer**: unemployment trend chart 2020-2024 added below sector chart.
- **Business Opportunity**: Smart Recommendations engine — user picks business type,
  priority (highest score / white-space / momentum / established), and region; top 5
  markets returned with a ranked bar chart.
- **Cluster Analysis tab**: KPI strip per cluster, interactive PCA scatter, cluster
  radar profile, filtered member table, download.
- **Sensitivity Analysis tab**: rank heatmap, rank volatility bar, weight scenario table, download.

### Documentation
- `README.md` fully rewritten to reflect all current capabilities.
- `CHANGELOG.md` created (this file).
